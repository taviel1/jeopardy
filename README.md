# jeopardy
This is my version of the game *Jeopardy!* (or *Svoya Igra*).

This is my version of the game *Jeopardy!* (or *Svoya Igra*). This small mini-game features a database of questions covering various topics and difficulty levels. Each time the game starts, five random topics and their corresponding questions are selected; the user can choose any question and then select one of four possible answers. The harder the question, the more points the player earns for a correct answer, while an incorrect answer yields zero points.
